
### Installing
Download repo as zip to /export/home/nz and unzip it

```
unzip predykcja.zip -d predykcja
```

Change directory to created folder

```
cd predykcja
```

Grant permission to init.sh

```
chmod +x init.sh
```
Run init.sh
```
./init.sh
```
Created db: predykcja
Crated tables: daneCO, daneCW
